Welcome to Logo Showcase for Elementor. With this awesome plugin you can build your logo showcase in 1 minute using Elementor. Main Features:
  • Addons For Elementor
  • Logo Carousel – Slider
  • Single control item: you can choose different settings (url, target etc) for each logo added in carousel/slider
  • Responsive: you can choose items to show in bases at device
  • Logo normal: you can create a simple logo with various settings
  • Custom Color
  • Animations with 200+ Effects
  • Translation Ready (include .po/.mo files)
Required:
  • Elementor installed

Installation

  1. Download the ZIP package from Codecanyon and Unzip it.
  2. Within the unzipped folder you will see a file named logoelementor.zip - this is the Plugin itself.
  3. Log into your Wordpress Dashboard and click on Plugins.
  4. Next click on Install Plugin then click on Upload.
  5. Click Add new -> Upload -> Browse then locate the file named logoelementor.zip and click Install Now.
  6. After the plugin has installed click on Activate Plugin and you are all set! Logo Showcase for Elementor is now installed!

ADD Elementor Widget

  1. Go to page / post and edit with Elementor Editor
  2. Find Logo Showcase choose your favorite block and drag and drop it inside your area
  3. Choose general options of selected block.
  4. After this save page
  5. Finish
